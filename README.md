# 🎓 Student Dashboard

A modern and responsive **Student Productivity Dashboard** built with **React** and **Tailwind CSS**. It helps students organize their academic life by managing tasks, tracking study sessions, calculating CGPA, setting goals, and saving useful learning resources—all in one place.

---

## ✨ Features

### 📊 Dashboard
- Overview of tasks, study hours, and current CGPA
- Monthly calendar widget
- Quick access to today's pending tasks
- Responsive dashboard layout

### ✅ Task Management
- Add, edit, and delete tasks
- Set task priority (High, Medium, Low)
- Mark tasks as completed
- Filter by All, Pending, and Completed
- Archive deleted tasks
- Local storage persistence

### 🎯 Goal Tracker
- Add weekly and monthly goals
- Track goal progress
- Mark goals as completed
- Delete goals
- Goal statistics

### 📚 Study Tracker
- Log study sessions
- Track today's and weekly study hours
- View total study sessions
- Delete study sessions

### 🎓 CGPA Calculator
- Add semester SGPA and credits
- Automatically calculate current CGPA
- Track total credits
- Delete semester records

### 📅 Calendar
- Full-year calendar view
- Monthly calendar widget for dashboard
- Year navigation

### 📖 Resources
- Save useful study resources
- Display website domain for cleaner UI
- Open resources directly
- Delete resources

### 📱 Responsive Design
- Mobile-friendly layout
- Responsive sidebar drawer
- Optimized for desktop, tablet, and mobile

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/student-dashboard.git
```

Navigate to the project

```bash
cd student-dashboard
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
